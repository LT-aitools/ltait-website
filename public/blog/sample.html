This is a Hidden Blog Post

Posted on: December 2024 | Category: Hidden

This is a sample hidden blog post that demonstrates how you can drop HTML files into the public/blog/ directory and they'll be accessible at /blog/filename (without the .html extension).

You can:

  • Add any HTML content you want
  • Include your own CSS styling
  • Embed videos, images, or other content
  • Create completely custom layouts

This route is not linked anywhere on your main site, so it's truly hidden unless someone knows the URL.
